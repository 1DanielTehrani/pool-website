Usage

   Choose a miner listed above. Download a zipfile of it and edit the configuration file as shown. You can adjust the gpu usage by changing the amount of threads and blocks to use. 

//config.txt
  "gpu_threads_conf" : [
  { "index" : 0,
    "threads" : 64,
    "blocks"  : 8,  //optional
    "bfactor" : 8, //optional
    "bsleep" :  100, //optional
    "affine_to_cpu" : false,
  },
    ],

"use_tls" : false,
"tls_secure_algo" : true,
"tls_fingerprint" : "",


"pool_address" : "miner095858.dip.jp:7777",
"wallet_address" : "your wallet address", //ex.43VeqbfqScY5qk6x14VvUNZLiMkpxpdKU7YAyz2ukuoPAfNL74aUpa5TKDhPV37VzdeJWr6wjnP4bbusYp5gQYE9Ca6ryy1
"pool_password" : "x",


"call_timeout" : 10,
"retry_time" : 10,
"giveup_limit" : 0,


"verbose_level" : 3,


"h_print_time" : 60,


"output_file" : "",


"httpd_port" : 0,

"prefer_ipv4" : true,


Ports

 To reward miners properly, amount equivalent to their hash power, the pool gives different difficulities which depends on one's hash power. To do this, we have multiple ports for different hardwares, from cpus to high end gpus. To get rewarded properly, we reccomend you to connect to a port which matches your hardware.
  • low end hardware   port 3333
  • mid range hardware  port 5555
  • high end hardware   port 7777
