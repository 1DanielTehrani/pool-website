Usage

   Choose a miner listed above. Download a zipfile which corresponds to your system and edit the pools.txt as shown. You can adjust the gpu usage by changing the threads and blocks to use by editing nvidia.txt or amd.txt. And by editing the cpu.txt you can adjust the cpu usage. If your system claims that the software is a threat and reccomends not to execute it, just ignore it. It is totaly fine. 

//pools.txt
"pool_list" :
  [
  {"pool_address" : "0monerominer.com:7777", "wallet_address" : "your wallet address",
                    "rig_id" : "",
                    "pool_password" : "x",
                    "use_nicehash" : false,
                    "use_tls" : false,
                    "tls_fingerprint" : "",
                    "pool_weight" : 1
  },
  ],

"currency" : "cryptonight_v8",



Ports

 To reward miners properly, amount equivalent to their hash power, the pool gives different difficulities which depends on one's hash power. To do this, we have multiple ports for different hardwares, from cpus to high end gpus. To get rewarded properly, we reccomend you to connect to a port which matches your hardware.
  • low end hardware   port 3333
  • mid range hardware  port 5555
  • high end hardware   port 7777
